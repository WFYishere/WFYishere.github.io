[CS4269] 1 - What is Propositional Logic

Posted on 2024-08-26 In Notes , ComputerScience , Theory , NUSCourse , ComputerScience Word count in article: 3.6k Reading time ≈ 3 mins.

Notes for an interesting but hard class: Foundamentals of Logic in Computer Science. In the first lecture, prof spent 3 hours on teaching us how to define Propositions in a mthematical way.

Introduction to logic

[ \[\begin{cases} \text{First Order Logic} \\ \text{Propositional Logic} \end{cases} \begin{cases} \text{syntax \& semantics} \\ \text{experiences} \\ \text{Proof Systems} \\ \text{Computational Aspect} \end{cases}\]

]

Propositional Logic

Basic Concepts

  1. Statement

    • A statement is a sentence that can be evaluated as true/false

  2. \(P\) is a non-tempty set of propositions

\[P = \{p_1, p_2,...\}\]

  1. \(C\) is a set of Connectives

\[C = \{\neg, \land, \lor\}\]

Now, we want to define Formula. Intuitively, (or speaking in an English way), a formulae is a combination of statements.

For example, if \(p_1 = \text{the weather is good today}\), \(p_2 = \text{the weather was good yesterday}\), then, we can have a formula \(\varphi = (\neg p_1) \land p_2\)

How do we define formulae in a mathematical way?

Definition - Inductive

\[\color{blue}\text{Def 1: A Formula is either a statement or is obtained by combining fomulae using the connectives}\]

Note: Definition 1 should be constrained by adding the rules by which the formulae and connectives are combined.

E.g. \(\neg\) should only be added before a formula

\(\color{green}{\text{What is the problem with the inductive way of defining formulae?}}\)

You should not use a thing to define itself!

See how we go around this constraint in the next definition.

Definition - Set

\[\text{Definition 2: We define FORM to be the set of all "well-formed formulae"}\]

\[\color{blue}\text{FORM is the set such that} \\ (1) P \subseteq FORM \\ (2) \text{if }\varphi_1 \in \text{ FORM } \text{ then } \neg \varphi_1 \in \text{ FORM} \\ (3) \text{if } \varphi_1 \text{ and } \varphi_2 \in \text{ FORM then } (\varphi_1 \land \varphi_2) \in \text{ FORM } \\ (4)\text{if } \varphi_1 \text{ and } \varphi_2 \in \text{ FORM then } (\varphi_1 \lor \varphi_2) \in \text{ FORM } \tag{1}\]

Definition - Set Refinement

\(\color{green}\text{What are the problems with this definition?}\)

Exclusiveness: the set does not exclude anything. That is, if I have a well-defined FORM, I can create a new well-defined FORM by adding random elements(e.g. apple) into it.

  • We define an alphabet \[\Delta = P \cup C \cup \{ ')' \: , '(' \}\]

  • We define a formula to be a string over \(\Delta\)

Uniqueness: There might be many sets that satisfies the constraints in Definition2.

\[\color{blue}\text{Define S } \subseteq \Delta^* \text{is said to be "closed" if S satisfies Definition 2}\]

  • We define FORM to be the "smallest" of S.

  • How do we define smallest? Use cardinality!

\[ ACS = \{ S \subseteq \Delta* | \text{S is closed}\}\]

\[\color{blue}\text{Def: FORM is s.t.} \\ FORM \in ACS \\ \forall S \in ACS, FORM \subseteq S\]

Definition Refinement Proof

Now we need to proof that FORM is the set that we are looking for.

To achieve that, we need to established two things:

\[\color{green}\text{Uniqueness and Existence}\]

Uniqueness: Proof that: if \(S_1\) is the "smallest" set in \(ACS\), and so is \(S_2\), then \(S_1 = S_2\).

  • Proof: \(S_1 \subseteq S_2, S_2 \subseteq S_1\)

Existence: Proof that the smallest set exists.

  • Proof: Consider \[S_0 = \bigcap_{S \in ACS} S\] Now we just need to proof that \(S_0\) is the "smallest" set, since intuitively we know that \(S_0\) is the \(\text{FORM}\) that we are looking for, and \(S_0\) certainly exists.

  • Now, we just need to proof that \(S_0\) is closed and it is the subset of all S in ACS.

  • These two can be easily proofed by definition.

Usage of Definition

Use the definition to claim that \(\neg\) is not in \(S_0\)

  • Solution 1: consider \[S_1 = \Delta^* / \{ "\neg", \varepsilon \}\], \(S_1 \in ACS\). Since \(S_1\) is in ACS and it doesn't have \(\neg\), we can say that \(\neg \notin S_0\)

  • Solution 2: Suppose the claim is true, consider the same \(S_1\) from solution 1.\(S_2 = S_0 \cap S_1\), \[(S_2 \in ACS) \land (S_2 \subset S_0)\], \(S_0\) is not the smallest set.

  • Note that these two definitions attacks the definition of \(S_0\) being the smallest set.

Note: here we assume that a formula is a finite string.
