[LeetCode] 152 - Maximum Product Subarray

Posted on 2024-05-31 Edited on 2024-06-02 In Solutions , LeetCode , Medium Word count in article: 2.8k Reading time ≈ 3 mins.

Easy problem but took me soooo long!

Problem Description

Given an integer array nums, find a subarray that has the largest product, and return the product.

The test cases are generated so that the answer will fit in a 32-bit integer.

\(1 <= nums.length <= 2 * 10^4\)

\(-10 <= nums[i] <= 10\)

Idea & Code

Idea: For any array,. the absolute value of the product is always increasing. Thus, the maximum array would have the following two features the maximum of all arrays that

  1. product is positive (negative for special cases)

  2. one end of the array is the "edge" of the array (we define edge as either the starting/ending point of the array or i where nums[i] = 0)
